# Código generado
## [HealthKitQuickStart](https://github.com/next-adrianvegas/Poc-HealthKit/tree/master/code_examples/HealthKitQuickStart) 
Este proyecto se ha realizado siguiendo el tutorial :  [https://www.raywenderlich.com/5045-healthkit-quick-start](https://www.raywenderlich.com/5045-healthkit-quick-start) 

Se trata de una aplicación que busca ayudar a los usuarios a consumir  el agua necesaria para mantenerse hidratado a lo largo del día.

En el se cubre la integración del framework en una App, configurando la petición de permisos necesaria y las principales funciones del framework, lectura de datos ( perfil de usuario ), escritura de datos (agua consumida), y la búsqueda de estos datos (cálculo del total de agua consumida hoy).

[Imágenes demo](https://github.com/next-adrianvegas/Poc-HealthKit/tree/master/code_examples/HealthKitQuickStart/demo_screens)

##  [HealthKitWorkoutApp](https://github.com/next-adrianvegas/Poc-HealthKit/tree/master/code_examples/HealthKitWorkoutApp) 
En este proyecto se ha desarrollado una aplicación para Apple Watch,  que registra  sesiones de ejercicio de los usuarios, mostrando los kilómetros recorridos, las calorías en función del tipo de ejercicio (Correr, nadar, ir en bici) y el ritmo cardiaco .

Con esta aplicación se han probado la configuración de permisos necesaria para el uso de los datos de usuario de la aplicación Health, la lectura, escritura y actualización de datos en la base de datos de Health y el uso de intervalos para medir estos datos. 

[Imágenes demo](https://github.com/next-adrianvegas/Poc-HealthKit/tree/master/code_examples/HealthKitWorkoutApp/demo_screens)
